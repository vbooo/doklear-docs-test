# Sécurité du widget

## Introduction
Le widget Doklear est conçu pour être embarqué sur des sites clients.
Vous devez limiter son usage aux domaines autorisés.

## Instructions
1. Définissez la liste des domaines autorisés dans le token widget.
2. Désactivez le token si vous suspectez un abus.
3. Surveillez les questions fréquentes dans l’analytics.

## Conclusion
Restreignez l’usage du widget aux domaines attendus.
Révoquez les tokens non utilisés ou partagés par erreur.
