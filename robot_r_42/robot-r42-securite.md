# Sécurité du robot R-42

## Prérequis

- Gants isolants
- Zone dégagée
- Robot hors tension

## Installation

1. Coupez l’alimentation.
2. Débranchez le câble principal.

## Configuration

1. Vérifiez l’état des connecteurs.
2. Replacez le capot.

## Conclusion

Ces précautions limitent les risques.
