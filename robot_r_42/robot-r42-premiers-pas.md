# Premiers pas avec le robot R-42

Bienvenue sur le robot R-42 , ce guide va vous expliquer comment démarrer rapidement.

## Prérequis

Vous devez avoir :
- une alimentation secteur
- un câble USB-C
- un accès administrateur au système .

## Installation

Vous devez brancher le robot sur une prise secteur puis attendez que le voyant devienne vert .

## Configuration

Tu dois configurer la puce pour que le robot fonctionne. Vous devez ensuite redémarrer le robot.

## Conclusion

Tu as maintenant toutes les informations nécessaires pour bien démarrer.
