# Réinitialisation du robot R-42 — Procédure A

## Prérequis

- Accès administrateur au panneau
- Robot branché sur secteur
- Connexion réseau disponible

## Procédure

1. Ouvrez le panneau de contrôle du robot.
2. Allez dans **Paramètres** puis **Maintenance**.
3. Cliquez sur **Réinitialiser**.
4. Confirmez l’opération et attendez la fin de la procédure.
5. Redémarrez le robot et vérifiez que le voyant devient vert.
6. Si le problème persiste, consultez les journaux système.

## Conclusion

Cette procédure restaure une configuration saine.
