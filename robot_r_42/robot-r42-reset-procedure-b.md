# R-42 — Remise à zéro (Procédure B)

## Prérequis

- Droits admin sur l’interface
- Alimentation électrique active
- Réseau accessible

## Étapes

1. Accédez à l’interface de gestion du robot.
2. Ouvrez le menu **Maintenance** dans la section **Settings/Paramètres**.
3. Lancez l’action **Reset / Réinitialisation**.
4. Validez, puis patientez jusqu’à la confirmation de fin.
5. Relancez le robot et contrôlez que l’indicateur passe au vert.
6. En cas d’échec, inspectez les logs.

## Conclusion

La remise à zéro permet de repartir d’un état stable.
